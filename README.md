# drippy-bot
Drippy's Discord bot

## Dependencies
- [MediaServer](https://github.com/DrippyMusic/MediaServer.git)
- [ffmpeg](https://www.ffmpeg.org/download.html/)

## Environment Variables
DISCORD_TOKEN=  
DRIPPY_API_TOKEN=  
DRIPPY_PUBLIC_KEY=  

## Run
`$ npm run dev`

## Build
`$ npm run build`
